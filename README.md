# Cantareira HardTour

Site institucional do projeto **Cantareira HardTour**, voltado para divulgação de **passeios off-road na região da Serra da Cantareira**.

O objetivo do site é apresentar a experiência, mostrar informações sobre os passeios e permitir que interessados conheçam melhor a proposta do projeto.

---

## Tecnologias utilizadas

Este projeto utiliza tecnologias modernas de desenvolvimento web:

* **Next.js**
* **Node.js**
* **HTML**
* **CSS**
* **JavaScript**

---

## 📂 Estrutura do projeto

A estrutura principal do projeto segue o padrão de aplicações **Next.js**:

```
/app ou /pages   → páginas do site
/public          → imagens e arquivos estáticos
/styles          → arquivos de estilo
/package.json    → dependências do projeto
```

---

## ▶️ Como rodar o projeto localmente

1. Clone o repositório

```
git clone https://github.com/studdw/Cantareira-HardTour.git
```

2. Entre na pasta do projeto

```
cd Cantareira-HardTour
```

3. Instale as dependências

```
npm install
```

4. Inicie o servidor de desenvolvimento

```
npm run dev
```

5. Abra no navegador

```
http://localhost:3000
```
